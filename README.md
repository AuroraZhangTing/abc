# 产学研项目管理系统

这是一个轻量、可直接运行的产学研项目管理系统，采用 `Node.js + HTML/CSS/JavaScript + 本地 JSON 存储`，不依赖数据库。

## 功能

- 项目管理
- 合同管理
- 经费认领
- 发票管理
- 项目结题
- 管理员面板
- 文件上传与下载
- 提交锁定与管理员重置
- 首次初始化管理员

## 目录

- `server.js` - Node.js 服务和 JSON 存储 API
- `public/index.html` - 单页前端
- `data/store.json` - 本地数据文件，首次运行自动生成
- `uploads/` - 上传文件目录

## 启动

1. 安装 Node.js 22
2. 进入项目目录
3. 执行：

```powershell
npm start
```

4. 打开：

```text
http://localhost:3000
```

## 首次使用

如果系统里还没有用户，页面会显示“系统首次初始化”。

- 输入管理员账号
- 输入管理员姓名
- 点击“创建首个管理员”

创建后，管理员可以新增项目、维护用户、填写全部流程并提交。

## 部署

### Bonto

适合本项目的免费部署方式。因为本系统使用本地 JSON 文件保存数据，Bonto 这类保留文件系统的 Node 部署更合适。

部署方式：

1. 将代码推到 Git 仓库
2. 在 Bonto 创建应用并连接仓库
3. 启动命令使用：

```bash
node server.js
```

4. 部署完成后通过 Bonto 提供的公网地址访问

### Vercel

Vercel 可以部署前端和无状态接口，但**不适合本项目的本地 JSON 持久化**，因为它的运行环境不适合长期保存本地文件。若要上 Vercel，建议改成外部存储或云数据库。

## 说明

- 普通用户提交后不可修改
- 管理员可编辑、删除、重置阶段
- 文件上传后统一保存在 `uploads/`
- 刷新后数据不会丢失，只要宿主平台保留本地文件
